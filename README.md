## Get Started

1. Clone to this repository:

```bash
  git clone https://github.com/sszobaer/sszobaerahmed.dev.git
  cd sszobaerahmed.dev
```

2. Install Dependencies:

```bash
yarn
```

3. Dev Run:

```bash
yarn dev
```

4. Build:

```bash
yarn build
```
